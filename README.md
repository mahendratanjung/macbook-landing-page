## <a name="introduction">âœ¨ Introduction</a>

A modern MacBook Pro landing page built with React, Three.js, GSAP, and TailwindCSS. It combines an interactive 3D product viewer, cinematic video sections, and scroll-driven animations to recreate the feel of Apple's product marketing pages.

Users can explore the MacBook in 3D, switch between 14" and 16" models, change colors, and discover key hardware and AI features through smooth, immersive sections.

## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally.

### Prerequisites

Make sure you have the following installed:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

### Cloning the Repository

```bash
git clone https://github.com/mahendratanjung/macbook-landing-page.git
cd macbook-landing-page
```

### Installation

Install the project dependencies using npm:

```bash
npm install
```

### Running the Project

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the project.
